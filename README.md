Создано кривыми руками. Созданно образовательных целей. Кто хочет используйте! не какие риски на себя не беру)))

Запускалось на севере linux ubuntu 22.04 -> в теории должно работать на версиях старше


Админ-панель: http://ВАШ_IP_СЕРВЕРА:5001
Тесты для пользователей: http://ВАШ_IP_СЕРВЕРА:5000


Что нужно сделать перед запуском:

sudo apt update
sudo apt install python3-pip -y
pip3 install flask

Для запуска лучше делать вот так:

nohup python3 admin.py > admin.log 2>&1 &

nohup python3 client.py > client.log 2>&1 &

Чтобы убить процесс:

Найти процесс админ-панель:

ps aux | grep admin.py

Убить процесс (замените 12345 на ваш PID)

kill 12345

Найти процесс пользователькие тесты:

ps aux | grep client.py

Убить процесс (замените 12345 на ваш PID)

kill 12345
